# Overview

[!INCLUDE[preview-note](./includes/preview-note.md)]

Bing Ads Scripts enables programmatic control of your Bing Ads data using JavaScript in Bing Ads' browser-based script editor.  Scripts can automate common data workflow interactions and manage repetitive tasks.  With Bing Ads Scripts you can quickly and easily integrate with Bing Ads without the need for a significant up front financial investment.
